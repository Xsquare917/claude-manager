# v1.2.0 更新日志

发布日期: 2026-01-29

## 新功能

### 1. 版本更新检测
- 自动检查 GitHub Releases 获取最新版本
- 检测到更新后弹出更新说明弹窗
- 左上角显示"新版本"徽章

### 2. 常用指令模板
- 设置页面新增"常用指令模板"
- 创建新会话时自动填入模板内容

### 3. 主题切换优化
- 深色/浅色滑动开关条
- 新增"跟随系统"选项
- 主题切换即时生效，无需点击保存

### 4. 项目切换高亮动画
- 当前项目边框高亮
- 项目名称和会话名称放大动画

### 5. 状态提示动画
- 右上角状态从上方弹入弹出
- 3秒后自动隐藏
- 当前任务显示磨砂玻璃效果

### 6. 终端边缘渐变
- 终端边界到容器边界的渐变过渡
- 跟随主题切换

## 优化

### 1. 终端滚动优化
- 智能滚动：仅当用户在底部时才自动滚动
- 用户输入后自动回到底部

### 2. 终端聚焦优化
- 初始化后自动聚焦终端
- 点击终端区域自动聚焦

### 3. 批量刷新概括修复
- 串行处理摘要请求，避免 API 限流

## 升级说明

直接替换即可，无需迁移数据。设置会自动保留。
